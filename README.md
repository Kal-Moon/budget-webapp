# budget-webapp
# 파이썬 Flask로 만드는 가계부 웹앱
## 프로젝트 소개(Project introduction)
(가계부 웹앱을 만들기. Creating a household ledger website)
## 주요기능(Main functions)
-수입, 지출 등  가계부 기능(Household ledger functions such as income and expenses)
-수입, 지출에 대한 월별 결산, 총결산(웹앱 입력 시점부터)등 통계 기능(Statistical functions including monthly and total settlements (from the moment of entry in the web app) for income and expenses)
-글자 크기, 시작 주차 설정 등 설정 기능(Settings functions such as font size and start week settings)
## 사용 기술(Technology used)
-미정
## 화면 구성(Screen composition)
가계부- 수입/지출 입력 및 요약, 달력 뷰(날짜별 내역), 월별 수입/지출 요약
통계- 해당 월 수입 통계, 해당 월 지출 통계, 월 결산, 총 결산
설정- 글자 크기 조정, 시작일 설정, 추후 다른 기능 도입 예정
